# ConfigFiles Directory

This directory contains configuration files for various
applications and system components that should be copied to the
user's ~/.config/ directory to set up the rice environment.

## Subdirectories

### gtk-3.0/

GTK 3.0 theme settings for customizing GTK-based application
appearances.

### hypr/

Configurations for Hyprland window manager:

• hyprland.conf: Main window manager settings
• hyprlock.conf: Lock screen configuration
• hypridle.conf: Idle management settings
• hyprlock/assets/: Images and assets for the lock screen

### nvim/

Neovim editor configurations:

• Lua-based plugin setups and keymaps
• Color schemes and options
• Lazy plugin manager lock file

### ranger/

Ranger file manager configurations:

• Commands, rifle associations, and scope settings

### wayland/

Wayland-specific configurations:

• Fastfetch system info display
• Kitty terminal emulator settings
• Mako notification daemon config
• Waybar status bar configs and styles
• Tmux terminal multiplexer settings
• Zsh shell with Starship prompt
• Scripts for various utilities (audio switcher, wallpaper
selector, etc.)
• Rofi themes for menus
• Current rice and wallpaper state files

### gtkrc

GTK resource file for additional GTK settings.
